\# IntelCrop local GIS data



Questa cartella contiene dataset GIS locali usati per sviluppo e ingest.

data/raw/landcover/calabria/olive\_pure/

Questi layer descrivono tipologie di impianto, non cultivar.

Non dedurre mai la cultivar dal layer CUT.
